Class: Note

Note()

Represents note values. Midi note numbers go from 0 to 127. A4 is tuned to 440Hz, and corresponds to midi note 69. F(n) = 440 * (2^(1/12))^(n - 69) = 440 * 2 ^ ((n-69)/12)
